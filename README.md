# Xennis's Dotfiles

## Installation

```sh
git clone --recurse-submodules https://github.com/Xennis/dotfiles ~/dotfiles
cd ~/dotfiles
./setup.sh
```

#### Dependencies

* awesome: chromium, termite
* git: vim, meld
* zsh: powerline-fonts
